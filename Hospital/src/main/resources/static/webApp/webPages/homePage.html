"!DOCTYPE html" in very begining before html headder.

"html" encloses whole page.

"head" un displayed meta data, aka tab of browser.

"title" is title.

"body" is main components.

"p" for paragraph sections.

"h1...h7" for headders lower number larger headder.

"br" for line breaks.

"strong" for bold, "em" for italics, "u" for underline.

"img src = "file_name" alt = "text if there is an error with image"" to place image.

"a href = "URL" target = "_blank"" a for anchor (beginning of hypertext link) href which specifies a location target is how it opens the page (replaces current page by default) >what to click to go to link . Then close "/a".

"link rel = "stylesheet" type = "test/css" href = "style.css"" link our css style to our html file.

In .css text file tags can have their styles changes. Examples are font-color: "color", font-familt:: "font style", etc. Multiple can be combined eg. h1, h2 { ... }.

when making tags e.g. "p" for paragraph sections it is good practice to name them as a class i.e. "p class= "name_of_class" so you can easily manipulate in css file. In css file use .name_of_class{ ... }. id can also be used but is bad practice and in css file its .name_of_id { ... }.

use "* { margin: 0; padding: 0; box-sizing: border-box; }" to start with a 0 border baseline and then edit the values induvidually.

"div" stands for divde to section out the page into boxes.

"margin-left: auto; margin-right: auto;" makes both margins even with the container size.

In the div classes width %, float for positioning, .clearfix:after {content: ""; display: table;clear: both;} so next div box isnt affected by floats. "margins = 20px auto 0 auto;" used for spacing. padding also used to space boxes. and "border = 1px solid rgb" for lines.

.author-box img { ... } makes all img inside that div box follow that css file style.

set parent div to relative and a child to absoluto to get things where you want them.

position: absolute; width: 1550px; top: 50%; left: 50%; transform: translate(-50%, -50%); to get a box centered.

background-size: cover; background-position: center; height: 100vh; for full background image to cover webpage.

background-image: linear-gradient(color 1, color 2), url(image_destination); to put a "filter" on background.

inline an element to inline block allows for padding and margins, e.g. inline element such as "a" can be changed to an inline block.

btn:link "link" means pre visited, and .btn:visited "visited" means clicked.

.btn:hover and .btn:active for hovered and clicked.

When making links to go to other sites use "ul" unsorted list which is a rel that we downloaded and href to for lists, and inside make seperate "li" elements with "a href" elements inside. It goes ul>li>a href.
Inside cascading lists you use one "ul" class name then just type the element after in order to edit the element. E.g .main-nav {...} .main-nav li {...} .main-nav li a:link {...}.



div class "col span-x-to-y" uses imported grid to organize and space the columns evenly.
